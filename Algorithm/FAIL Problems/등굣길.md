좋아! 프로그래머스의 **"등굣길"** 문제는 전형적인 **경로 탐색 DP 문제**야.  
먼저 문제 개념부터 짚고, 그다음에 **DP 식 세우는 흐름**을 차근차근 같이 정리해볼게!

---

## ✅ 문제 요약

- `m` x `n` 격자 (1,1)에서 (m,n)까지 가야 함.
- **오른쪽** 또는 **아래쪽**으로만 이동 가능.
- **물에 잠긴 지역(lost)**은 못 지나감.
- **경우의 수**를 **1,000,000,007로 나눈 나머지**를 반환해야 함.

---

## 🧠 DP 개념 적용

### 1. ✅ **DP 배열 정의**

```python
dp[y][x] = (1,1)에서 (x,y)까지 올 수 있는 경로의 개수
```

> (1,1)이 시작이고, 문제는 좌표가 (x, y) 기준이라 조금 헷갈릴 수 있지만,
> 우리는 `dp[y][x]`를 사용한다고 생각하면 돼. (보통 배열 인덱스는 row, col 순이니까!)

---

### 2. ✅ **점화식 세우기**

현재 위치 `(y, x)`까지 오는 방법은:
- **위에서 내려오거나** `dp[y-1][x]`
- **왼쪽에서 오는 경우** `dp[y][x-1]`

```python
dp[y][x] = dp[y-1][x] + dp[y][x-1]
```

단, `물에 잠긴 지역`은 `dp[y][x] = 0`

---

### 3. ✅ **초기값 설정**

- `(1,1)`은 무조건 시작점 → `dp[0][0] = 1`
- 첫 번째 행과 첫 번째 열은 **이전 칸이 막혀있지 않다면** `1`로 채움
  - 막힌 이후부터는 모두 `0`으로 채워야 함

---

### 4. ✅ 코드 예시 (Python)

```python
def solution(m, n, puddles):
    MOD = 1_000_000_007
    dp = [[0] * m for _ in range(n)]

    for i in range(n):  # 세로줄 반복 (행)
        for j in range(m):  # 가로줄 반복 (열)
            if dp[i][j] == -1:
                # 물에 잠긴 지역이면 경로가 아예 불가능하므로 0으로 설정하고 넘김
                dp[i][j] = 0
                continue

            if i == 0 and j == 0:
                # 시작점 (맨 좌측 상단): 경로 1가지 (자기 자신)
                dp[i][j] = 1
            else:
                # 위쪽에서 내려오는 경우가 존재하면 더함
                if i > 0:
                    dp[i][j] += dp[i-1][j]

                # 왼쪽에서 오는 경우가 존재하면 더함
                if j > 0:
                    dp[i][j] += dp[i][j-1]

            # 문제에서 요구한 대로 1,000,000,007로 나눈 나머지를 저장
            dp[i][j] %= MOD


    return dp[n-1][m-1]
```

## 📌 정리 요약

| 단계 | 설명 |
|------|------|
| 상태 정의 | `dp[y][x] = (1,1) → (x,y)`까지 가는 경로의 수 |
| 점화식 | `dp[y][x] = dp[y-1][x] + dp[y][x-1]` |
| 초기값 | `dp[0][0] = 1` + 경계선 처리 |
| 예외처리 | 물에 잠긴 곳은 `dp[y][x] = 0` |
