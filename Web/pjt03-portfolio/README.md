### 후기
그동안 CSS를 어렵다고 느껴 연습할 엄두가 나지 않았다.
이번 기회에 제대로 해보자고 마음을 먹고 html과 css를 연습했다.
그동안은 허우적대며 제대로 찾지 못했지만 이젠 척척 클래스도 만들고 이리저리 움직이며 많이 익숙해졌다. 영화 프로젝트는 진행하지 못했지만 그래도 이번 포트폴리오를 만들면서 내 자신을 돌아보고 다른 사람들의 포트폴리오 사이트를 보며 더 많이 배우고 알아야 함을 느꼈다.
