# CLI (Command Line Interface)

> GUI는 CLI에 비해 사용하기 쉽지만, 단계가 많고 성능을 상대적으로 더 많이 소모함

> 가장 중요한건 내가 어디 있는지(경로) 알아야 한다!
## 1. 명령어

- ***.*** 은 현재 디렉토리
- ***..*** 은 현재 상위 디렉토리(부모 폴더)
- ***cd:*** 현재 작업 중인 디렉토리를 변경 (위치 이동)
- ***start:*** 폴더/파일을 열기
- ***rm:*** 파일 영구 삭제 (-r 옵션을 사용해 디렉토리 삭제 / tab을 통해 자동완성기능)
- ***pwd***
- ***mkdir***
- ***ls***
- ***touch***
- ***mv***
- ***ctrl + l***
- ***절대 경로***
- ***상대 경로***
